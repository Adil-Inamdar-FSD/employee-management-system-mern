# Employee Management System MERN

A full-stack Employee Management System built with the MERN stack.  
This project helps manage employees, attendance, leave requests, payslips, profiles, and admin dashboard features.

## 🚀 Features

- Admin Dashboard
- Employee Management
- Attendance Tracking
- Leave Management
- Payslip Management
- Profile Settings
- Login System
- Responsive UI
- Full-stack MERN structure

## 🛠️ Tech Stack

### Frontend

- React.js
- React Router DOM
- CSS / Tailwind CSS
- Lucide React Icons

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt.js
- CORS
- Dotenv
